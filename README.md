2. 開発背景
解決したい課題: 幼児期における性被害の防止や、自分の体を大切にする意識（ボディリスペクト）の向上。

ターゲット: 3歳〜6歳の子どもとその保護者。

こだわり: 専門知識を背景にしつつ、子供が怖がらずに遊べる「ゆるキャラ」を活用したUI/UX。

3. 使用技術（Tech Stack）
スキルシートの内容を裏付ける技術一覧です。

Backend: PHP 8.4 / Laravel 12.51

Frontend: Blade / JavaScript (Vanilla JS)

Infrastructure: Docker / Render / GitHub Actions (CI/CD)

Database: SQLite

4. 主な機能
水着着せ替えゲーム: ボタン操作でキャラクターに水着やマスクを着せ、プライベートゾーンを学習する。

レスポンシブ対応: 親子でスマホを触りながら学習できるデザイン。
